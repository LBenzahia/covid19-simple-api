# covid19-simple-api
A simple api to return Confirmed, Deaths and Recovered history for a country

The data are from [CSSE](https://github.com/CSSEGISandData/COVID-19) and 
[coronatracker](https://www.coronatracker.com/analytics/)

## Endpoints

``/stats``

``/history``
